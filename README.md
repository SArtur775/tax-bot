# 💰 Налоговый Помощник - Telegram Bot

Умный бот для налоговых расчетов и консультаций в России.

## 🚀 Функциональность

- 📊 Калькуляторы налогов (НДФЛ, УСН, Самозанятые)
- 💰 Налоговые вычеты
- 📅 Напоминания о сроках
- 👤 Помощь самозанятым
